# Snake Game

A simple snake game built with Python's built-in `turtle` module.

## How to Run

```
python snake.py
```

No external packages required.

## Controls

| Key     | Action |
|---------|--------|
| Up / W  | Move up    |
| Down / S| Move down  |
| Left / A| Move left  |
| Right / D| Move right |

## Rules

- Guide the snake to eat the red food to grow and earn points.
- Avoid hitting the walls or your own tail.
- The game gets faster as your score increases.
- Your high score is saved during the session.
